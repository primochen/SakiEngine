# 睁眼转场效果 (Blink Transition)

## 简介

睁眼转场效果模拟了人睁开眼睛的视觉效果：上下两条黑色遮罩从中间向上下移开，呈现出一个从黑屏到新场景的流畅过渡。

## 使用方法

在 `.sks` 脚本中，使用 `with blink`、`with eyeopen` 或 `with eye` 来应用睁眼转场效果：

```sks
// 基本用法
scene sky with blink

// 可用的别名
scene home with eyeopen
scene school with eye
```

## 效果说明

**睁眼转场分为两个阶段：**

1. **闭眼阶段（前50%）**：上下黑色遮罩从屏幕边缘向中间合拢，逐渐覆盖整个屏幕
2. **睁眼阶段（后50%）**：黑色遮罩从中间向上下边缘移开，露出新场景

**时间点：**
- 场景切换发生在动画的中点（50%），此时屏幕完全被黑色遮罩覆盖

## 示例脚本

```sks
label morning_wake_up
scene bl

// 主角从睡梦中醒来
"我缓缓睁开双眼..."
scene bedroom with blink

"阳光洒进房间，新的一天开始了。"

// 回忆场景
"想起昨天的事情..."
scene flashback with blink

"那时候的我还不知道..."

return
```

## 适用场景

睁眼转场效果特别适合以下情境：

- 💤 角色从睡眠中醒来
- 👁️ 角色睁开眼睛或恢复意识
- 🔄 从回忆/梦境回到现实
- 🎬 需要强调"视角切换"的场景
- ⚡ 失去意识后苏醒的情节

## 与其他转场效果对比

| 转场类型 | 关键词 | 视觉效果 | 适用场景 |
|---------|--------|---------|---------|
| 淡入淡出 | `fade` | 黑屏渐变 | 通用场景切换 |
| 溶解 | `diss` | 图片直接渐变 | 平滑的场景过渡 |
| 擦除 | `wipe` | 旋转扇形擦除 | 时空转换、特效 |
| **睁眼** | `blink` / `eyeopen` / `eye` | **上下遮罩移开** | **醒来、恢复意识** |

## 技术细节

- **持续时间**：默认 800ms（可通过引擎配置调整）
- **曲线**：使用 `Curves.easeInOut` 实现平滑过渡
- **实现**：通过 `CustomPainter` 绘制上下两个矩形遮罩
- **性能**：轻量级实现，无额外资源加载

## 调试信息

在控制台中可以看到转场日志：
```
[BlinkTransition] 到达转场中点，执行回调
```

这表示场景切换已在转场动画的中点执行。
